# **Редактор списка задач.**

### Возможности:

1. добавление, редактирование, удаление, просмотр задач
2. добавление, редактирование, удаление, просмотр тегов и цветных иконок
3. фильтр по тегу
4. данные зранятся в json(добавление, изменение и удаление)
5. роутинг RRDv6

### Запуск:
**npm start**

**npm run serve**
